# Output from Template Designer for template-categories

Here are the wireframe styling guidelines for the three core theme categories:

**1. "The Developer"**

* **Overall Structure:**
	+ Header: Terminal-inspired title bar with a bold font and a subtle gradient effect
	+ Hero Section: A large, centered code snippet or a screenshot of a project's codebase
	+ Portfolio Grid: A responsive grid system featuring individual project cards with a focus on GitHub graphs (e.g., commit history, issue tracking)
	+ Skills/Tools Section: A list or grid showcasing the developer's programming languages, frameworks, and tools
* **Color Scheme:**
	+ Primary Color: #2E4053 (a deep blue reminiscent of terminal screens)
	+ Secondary Color: #F7F7F7 (a light gray for background elements and text)
	+ Accent Color: #66D9EF (a bright green for GitHub graphs and highlights)
* **Fonts:**
	+ Primary Font: Open Sans or a similar sans-serif font
	+ Code Snippet Font: A monospaced font like Consolas or Courier New

**2. "The Artist"**

* **Overall Structure:**
	+ Header: A visually striking title treatment with a bold, artistic font and a splash of color
	+ Hero Section: A large, centered image or a gallery grid showcasing the artist's work (e.g., paintings, photographs)
	+ Portfolio Grid: A responsive grid system featuring individual project cards with focus on visual embeds (e.g., images, videos)
	+ About/Story Section: A brief bio and statement about the artist's creative process
* **Color Scheme:**
	+ Primary Color: #3498DB (a calming blue-green hue)
	+ Secondary Color: #FFFFFF (white for background elements and text)
	+ Accent Color: #E74C3C (a deep orange-red for highlights and emphasis)
* **Fonts:**
	+ Primary Font: Merriweather or a similar serif font
	+ Visual Embed Fonts: A clean, legible sans-serif font like Helvetica or Arial

**3. "The Streamer"**

* **Overall Structure:**
	+ Header: A bold, neon-lit title treatment with a futuristic feel
	+ Hero Section: A large, centered screenshot of a gaming session (e.g., Twitch stream)
	+ Portfolio Grid: A responsive grid system featuring individual project cards with focus on Twitch embeds (e.g., streams, highlights)
	+ About/Community Section: A brief bio and information about the streamer's community
* **Color Scheme:**
	+ Primary Color: #F7DC6F (a bright, energetic yellow-orange)
	+ Secondary Color: #333333 (dark gray for background elements and text)
	+ Accent Color: #9B59B6 (a deep purple-blue for highlights and emphasis)
* **Fonts:**
	+ Primary Font: Arial or a similar sans-serif font
	+ Twitch Embed Fonts: A clean, legible sans-serif font like Helvetica or Arial

For each theme, I've defined the overall structure, color scheme, and fonts to create distinct visual identities that resonate with their respective audiences. The sub-designers (Portfolio Designer, Mobile Bio Designer) will work within these guidelines to develop custom templates for each category.
