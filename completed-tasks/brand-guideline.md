# Output from Creative Director for brand-guideline

As the Creative Director for getprofile.link, I'm excited to help establish our brand's ultimate "Elite" aesthetic rules. Below is a comprehensive document outlining our Tailwind utility tokens, defining the perfect blend of deep blacks, sleek whites, modern typography families, and accent colors that replicate the glassmorphism feel of our competitors.

**Brand Typography Tokens:**

1. **Primary Font Family:** Montserrat (Font Awesome's default font)
	* Use for headings, titles, and primary text
2. **Secondary Font Family:** Inter (a clean, sans-serif font with excellent readability)
	* Use for body text, descriptions, and secondary text
3. **Typography Settings:**
	+ Font size: 16px (default) - 24px (max)
	+ Line height: 1.5 (default) - 2 (max)
	+ Letter spacing: 0.25em (default) - 0.5em (max)

**Color Tokens:**

1. **Primary Color:** #212121 (a deep, rich black)
	* Use for backgrounds, borders, and primary text
2. **Secondary Color:** #FFFFFF (pure white)
	* Use for highlights, hover effects, and secondary text
3. **Accent Colors:**
	+ Glassmorphism-inspired green: #34C759 (a muted, earthy tone)
		- Use for buttons, icons, and call-to-actions
	+ Soft blue: #87CEEB (a calming, gentle hue)
		- Use for links, hover effects, and subtle highlights
4. **Background Gradients:**
	+ Main gradient: linear-gradient(to bottom, #212121 50%, #34C759 50%)
		- Use as a background or overlay to add depth and dimension

**Additional Tokens:**

1. **Border Radius:** 0.5em (default) - 2em (max)
2. **Box Shadow:** rgba(0, 0, 0, 0.25) 0px 4px 8px
3. **Transition Duration:** 200ms

**Best Practices:**

1. Always use our primary font family for headings and titles.
2. Use our secondary font family for body text and descriptions.
3. Apply the glassmorphism-inspired green accent color to buttons, icons, and call-to-actions.
4. Utilize the soft blue accent color for links, hover effects, and subtle highlights.
5. Keep typography settings consistent throughout the platform.
6. Use the primary and secondary colors wisely to maintain visual cohesion.

**Conclusion:**

By establishing these Tailwind utility tokens, we'll ensure a cohesive brand aesthetic across all themes and UI elements on getprofile.link. Our elite design will scream "premium" to users, setting us apart from competitors. Remember to follow best practices for using these tokens, and let's create an exceptional user experience!
