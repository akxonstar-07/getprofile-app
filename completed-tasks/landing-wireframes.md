# Output from Website Designer for landing-wireframes

Based on the task instructions, I've created a revised wireframe for the landing page that incorporates the new "Creator Features" roll-out and emphasizes the E-commerce and Media-kit portfolios available to Pro users.

**Landing Page Wireframes:**

The updated homepage is designed to showcase the new Creator Features and highlight the benefits of being a Pro user. The layout is divided into three main sections:

1. **Hero Section:**
	* A prominent header with the getprofile.link logo and tagline "Unlock Your Profile"
	* A hero image featuring a diverse group of creators showcasing their profiles
	* A call-to-action (CTA) button "Start Creating" that leads to the sign-up page

**E-commerce Portfolio Showcase:**

* A grid-based section featuring 4-6 tiles, each representing an e-commerce portfolio example (e.g., fashion, art, food)
* Each tile includes:
	+ An eye-catching image of the product
	+ The creator's name and profile picture
	+ A brief description of their work
	+ A "View Profile" button to learn more

**Media-kit Portfolio Showcase:**

* A staggered grid layout featuring 3-4 tiles, each representing a media kit portfolio example (e.g., photography, illustration, writing)
* Each tile includes:
	+ An eye-catching image or thumbnail of the work
	+ The creator's name and profile picture
	+ A brief description of their work
	+ A "View Profile" button to learn more

**Scroll-jacking Animation:**

To enhance user engagement and drive signups, I've designed a subtle scroll-jacking animation that reveals additional information as users scroll down the page. As they reach each portfolio tile, the following animations occur:

* E-commerce tiles:
	+ The product image zooms in slightly
	+ A brief quote or testimonial from the creator appears below the image
* Media-kit tiles:
	+ The thumbnail expands to show a larger preview of the work
	+ A short description of the project or its relevance to the user's interests appears above the thumbnail

**CTA and Navigation:**

* A prominent CTA button "Start Creating" at the bottom of the page, encouraging users to sign up for a Pro account
* A navigation menu with links to key sections (e.g., About, Features, Sign Up)

The updated wireframe is designed to effectively communicate the new Creator Features and drive user engagement through interactive animations. The E-commerce and Media-kit portfolio showcases provide an inspiring glimpse into the creative work of Pro users, while the scroll-jacking animation keeps users engaged and curious about what's next.
